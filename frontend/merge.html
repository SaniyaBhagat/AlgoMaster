Merge Sort

Definition

Merge Sort is a divide-and-conquer algorithm that splits the array into halves, recursively sorts them, and then merges the sorted halves into one.

Pseudocode

function mergeSort(arr):
    if length of arr > 1:
        mid = length of arr / 2
        left = arr[0...mid]
        right = arr[mid...end]
        
        mergeSort(left)
        mergeSort(right)
        
        merge(left, right, arr)

Time & Space Complexity

  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n log n)
  • Space: O(n) - not in-place

Real-World Applications

  • Used in external sorting (e.g., sorting data on disk).
  • Efficient for large datasets.
  • Stable and predictable performance.
