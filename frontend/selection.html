Selection Sort

Definition

Selection Sort is a simple comparison-based sorting algorithm that divides the array into a sorted and unsorted part.
It repeatedly selects the smallest (or largest) element from the unsorted section and swaps it with the first unsorted element.

Pseudocode

for i = 0 to n-1:
    min_index = i
    for j = i+1 to n:
        if arr[j] < arr[min_index]:
            min_index = j
    swap arr[i] with arr[min_index]

Time & Space Complexity

  • Best Case: O(n²)
  • Average Case: O(n²)
  • Worst Case: O(n²)
  • Space: O(1) – in-place sort

Real-World Applications

  • Simple sorting tasks for small datasets.
  • Good for teaching basic sorting logic.
  • Useful when memory writes are expensive (as it minimizes swaps).

Real-Life Analogy

Imagine selecting the shortest student in a line and moving them to the front. You then repeat the process for the rest of the line. That’s Selection Sort!
