Bucket Sort

Definition

Bucket Sort is a sorting algorithm that distributes elements into a number of buckets.
Each bucket is then sorted individually, either using a different sorting algorithm or recursively applying Bucket Sort itself.

Pseudocode

function bucketSort(arr):
    create n empty buckets
    for each element in arr:
        insert into appropriate bucket
    for each bucket:
        sort the bucket
    concatenate all buckets into arr

Time & Space Complexity

  • Best Case: O(n + k)
  • Average Case: O(n + k)
  • Worst Case: O(n²) – if all elements fall in one bucket
  • Space: O(n + k)

Real-World Applications

  • Sorting large datasets with uniformly distributed floating-point numbers.
  • Used in graphics and computational geometry.
  • Efficient for data with known distribution range.

Real-Life Analogy

Imagine sorting a group of students into bins based on height ranges (like 150-160 cm, 160-170 cm, etc.), then sorting each bin. This is essentially how Bucket Sort works!
