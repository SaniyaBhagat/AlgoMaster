Linear Search

Definition

Linear Search is the simplest searching algorithm that checks each element in a list sequentially until the target element is found or the list ends.

Pseudocode

function linearSearch(arr, target):
    for i = 0 to arr.length - 1:
        if arr[i] == target:
            return i
    return -1

Real-World Applications

  • Searching items in small unsorted lists.
  • Useful when list is short or unsorted and infrequently searched.
  • Checking presence of a value before performing an operation.
