Binary Search

Definition

Binary Search is an efficient algorithm for finding a target value within a sorted array.
It repeatedly divides the search interval in half and eliminates half of the remaining elements until the target is found.

Pseudocode

function binarySearch(arr, target):
    low = 0
    high = arr.length - 1
    while low ≤ high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        else if arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

Time & Space Complexity

  • Best Case: O(1)
  • Average Case: O(log n)
  • Worst Case: O(log n)
  • Space: O(1) – iterative

Real-World Applications

  • Looking up a word in a dictionary.
  • Searching for a name in a sorted contact list.
  • Finding records in a database using index-based search.

Real-Life Analogy

Imagine searching for a word in a dictionary: you don't go page by page—you open in the middle, check, and then go left or right depending on the word. That’s Binary Search in action!
